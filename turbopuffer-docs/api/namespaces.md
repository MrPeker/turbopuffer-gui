GET /v1/namespaces
------------------

Paginate through your namespaces.

### [](https://turbopuffer.com/docs/namespaces#parameters)Parameters

[](https://turbopuffer.com/docs/namespaces#param-cursor)

**cursor**string optional

retrieve the next page of results (pass `next_cursor` from the response payload)

* * *

[](https://turbopuffer.com/docs/namespaces#param-prefix)

**prefix**string optional

retrieve only namespaces that match the prefix, e.g. `foo` would return `foo` and `foo-bar`.

* * *

[](https://turbopuffer.com/docs/namespaces#param-page_size)

**page_size**string default: 1000

limit the number of results per page (max of 1000)

**Warning:** This endpoint is currently not consistent. In other words, it's possible for short periods of time that a namespace _exists_ but is not returned on this endpoint (or, that a namespace _doesn't exist_, but is returned from this endpoint).

Copy

```python
import turbopuffer

tpuf = turbopuffer.Turbopuffer(
    region='gcp-us-central1', # pick the right region: https://turbopuffer.com/docs/regions
)

# List all namespaces
namespaces = tpuf.namespaces()
for namespace in namespaces:
    print('Namespace', namespace.id)
```
